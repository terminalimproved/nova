# Nova - An AI Hub
Nova is an new project from mine to AI lovers. AI now run in your terminal, but on external servers from HuggingFace.

# How to install?
### Step - 01
git clone https://github.com/terminalimproved/nova
### Step - 02 
cd nova
### Step - 03 
pip install -r requirements.txt
### Step - 04 
python3 app.py
### Step - 05 
Be happy!
